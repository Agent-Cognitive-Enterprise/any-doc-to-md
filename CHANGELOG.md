# Changelog

All notable changes to ADTM will be documented in this file.

The project follows semantic versioning once public releases begin. Until then,
`0.1.0` tracks the pre-public release line.

## Unreleased

- Preparing ADTM for open-source release under Apache-2.0.
- Added open-source readiness tracking and public contribution/security docs.
- Set the default adapter set to `inhouse` only while keeping external
  adapters first-class optional adapters.
- Added dated benchmark documentation for the 2026-04-23 adapter corpus run.
- Added PyMuPDF classifier guardrails so users do not need `pymupdf-layout` to
  avoid noisy layout recommendations.

## 0.1.0

- Initial package version for document-to-Markdown conversion, QA, adapter
  tournaments, LLM-assisted audit helpers, and benchmark tooling.
